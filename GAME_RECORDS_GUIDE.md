## 🎮 快速上手

### 访问游戏记录
1. 登录账号（测试账号：admin/123456）
2. 点击导航栏右上角的用户菜单

## 📊 查看方式

### 方式一：查看所有游戏记录

**步骤：**
```
1. 游戏筛选器保持"全部游戏"
2. 表格显示所有游戏的混合记录
3. 可以看到每条记录的游戏名称、图标
```

**适用场景：**
- 浏览最近的游戏活动
- 查看所有游戏的总体表现
- 对比不同游戏的分数

---

### 方式二：查看单个游戏记录 ⭐推荐

**步骤：**
```
1. 点击游戏筛选器
2. 选择要查看的游戏（如"宝石消除"）
3. 自动显示该游戏的统计卡片
4. 表格只显示该游戏的记录

**显示内容：**

│   🎮  45     │ │   ⭐ 9999    │ │   📊 8500    │ │  ⏱️ 16小时   │
│  游玩次数    │ │   最高分     │ │   平均分     │ │   总时长     │
└──────────────┘ └──────────────┘ └──────────────┘ └──────────────┘
```

#### 记录表格（隐藏游戏名称列）
```
# | 得分  | 游戏时长  | 排名      | 游戏时间
1 | 9999  | 22分15秒  | 🥇 第1名  | 2024-11-10 13:20
2 | 8800  | 18分40秒  | 第5名     | 2024-11-09 15:45
3 | 8500  | 15分30秒  | 第8名     | 2024-11-08 20:10
```

**适用场景：**
- 深入分析某个游戏的表现
- 查看单个游戏的历史最高分
- 了解某个游戏的平均水平
- 统计某个游戏的投入时间

## 🎯 分数标签说明

系统会根据您在该游戏中的分数表现，自动标记不同颜色：

| 颜色 | 含义 | 条件 | 示例 |
|------|------|------|------|
| 🔴 红色 | **最高分** | 等于该游戏的历史最高分 | 9999 |
| 🟡 橙色 | **优秀** | 在前20%的高分范围 | 9500-9998 |
| 🟢 绿色 | **良好** | 在前50%的中等分范围 | 8000-9499 |
| ⚪ 灰色 | **普通** | 其他分数 | <8000 |

**示例：**
假设您在"宝石消除"中的分数分布是：
- 最高分：9999
- 最低分：5000
- 分数范围：4999

那么标签颜色会这样分配：
- `9999` → 🔴 红色（最高分）
- `9000-9998` → 🟡 橙色（前20%，范围约1000分）
- `7500-8999` → 🟢 绿色（前50%，范围约2500分）
- `5000-7499` → ⚪ 灰色（普通）

## 📈 如何分析游戏表现

### 1. 查看单游戏统计
```
选择游戏 → 查看统计卡片
- 游玩次数：了解投入程度
- 最高分：看历史最佳表现
- 平均分：评估稳定水平
- 总时长：统计时间投入
```

### 2. 对比不同游戏
```
切换不同游戏，对比统计数据：
- 哪个游戏玩得最多？（游玩次数）
- 哪个游戏分数最高？（最高分）
- 哪个游戏最稳定？（平均分）
- 哪个游戏投入最多？（总时长）
```

### 3. 分析进步情况
```
按时间查看记录：
- 近期分数是否提高？
- 排名是否上升？
- 平均分是否增长？
```

### 4. 找出薄弱环节
```
对比平均分：
- 平均分低的游戏需要练习
- 分数波动大的游戏不稳定
- 长时间低分需要改变策略
```

## 🔧 常见操作

### 操作1：查看记录详情
```
点击记录行的"查看详情"按钮
→ 弹出对话框显示完整信息
→ 包含游戏名、分数、时长、排名、时间
```

### 操作2：删除单条记录
```
点击记录行的"删除"按钮
→ 弹出确认对话框
→ 确认后从列表中移除
→ 统计数据自动更新
```

### 操作3：清空所有记录
```
点击筛选栏的"清空记录"按钮
→ 弹出确认对话框
→ 确认后删除所有记录
→ 无法恢复，请谨慎操作
```

### 操作4：搜索记录
```
输入关键词（游戏名或分数）
→ 选择日期范围（可选）
→ 点击"搜索"按钮
→ 显示符合条件的记录
```

### 操作5：重置筛选
```
点击"重置"按钮
→ 清空所有筛选条件
→ 显示全部记录
→ 分页重置为第一页
```

## ❓ 常见问题

### Q1: 为什么看不到统计卡片？
**A:** 统计卡片只在选择特定游戏时显示。选择"全部游戏"时不显示统计卡片。

### Q2: 如何查看某个游戏的平均分？
**A:** 选择该游戏，查看统计卡片中的"📊 平均分"。

### Q3: 分数标签颜色是什么意思？
**A:** 颜色表示该分数在该游戏中的水平：红色=最高，橙色=优秀，绿色=良好，灰色=普通。

### Q4: 如何知道自己哪个游戏玩得最多？
**A:** 依次选择不同游戏，对比"🎮 游玩次数"。

### Q5: 删除记录后可以恢复吗？
**A:** 不可以。删除操作是永久的，建议删除前仔细确认。

### Q6: 为什么有些游戏没有记录？
**A:** 因为还没有玩过该游戏，或者记录已被删除。

### Q7: 统计数据多久更新一次？
**A:** 实时更新。每次添加或删除记录后，统计数据会立即重新计算。
